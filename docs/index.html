Stay ahead of resets.

CodexBar keeps your session + weekly windows (and credits where available) in the menu bar, so you never have to guess when you’re safe to ship.

Download latest View on GitHub

Homebrew: brew install --cask steipete/tap/codexbar

  • One status item per provider (Codex, Claude, Cursor, Gemini, Antigravity, z.ai).
  • Session + weekly bars, plus reset times.
  • Web-backed limits via browser cookies (Codex, Claude, Cursor).
  • Minimal UI, no Dock icon.

Providers

Codex · Claude Code · Cursor · Gemini · Antigravity · z.ai

Data sources

CLI + local RPC where available, web dashboards via browser cookies for Codex/Claude/Cursor, plus z.ai API key support (z.ai dashboard).

Privacy

Reuses existing browser cookies (Safari/Chrome/Firefox) for dashboard access — no passwords stored. When cookies are missing, CodexBar falls back to local CLI output.
